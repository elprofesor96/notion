# Notion Widgets

Minimalist embeddable widgets for Notion.

## Widgets

- **Focus Timer** — Pomodoro timer with circular progress, 4 modes, and audio alerts
- **Daily Wins** — Daily checklist with animations and local persistence

## How to Use

1. Go to [notion.elprofesor.io](https://notion.elprofesor.io)
2. Toggle dark/light to match your Notion theme
3. Preview the widget or copy the embed URL
4. In Notion, type `/embed`, paste the URL, and you're done
